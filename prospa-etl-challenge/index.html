The Pipeline Architecture

We migrated from reporting based on 3rd Normal Form structures into a hyper-flattened Star Schema using PySpark's distributed engine.

Bronze Layer Raw delimited files ingested dynamically with Schema Inference, stripping trailing null delimiters.
Silver Layer Cleaned, filtered, and logically grouped. Appended Custom Revenue and Account Balance Classifications.
Gold Layer Massive Fact Table centralized around Flattened Dimensions for optimal BI querying.
